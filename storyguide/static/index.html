Help & About

How to run and use RoadTripper

What it does

RoadTripper watches your browser's live location, identifies nearby towns and notable places, generates short road trip commentary for children or adults, and can save each spoken item as searchable trip history.

Start the app

The backend runs locally at http://127.0.0.1:8000. You can run it manually from Terminal or keep it running with the installed macOS LaunchAgent. 

cd "/Users/jay/Documents/Playground 8"
zsh ./deploy_storyguide.sh

First use

Enter a trip name, pick a narration mode and age band, choose whether to save history and speak aloud, choose Live or Demo data mode, select a voice and optional free OpenRouter model if you want, then click Start Trip. Allow location access when the browser asks.

Interface tour

Current place shows the nearest town and facts. Nearby towns tracks towns near your last live location or map selection. Live narration shows spoken items, and Archive searches saved commentary.

Useful commands

launchctl print gui/$(id -u)/com.jay.storyguide
launchctl kickstart -k gui/$(id -u)/com.jay.storyguide
launchctl bootout gui/$(id -u) /Users/jay/Library/LaunchAgents/com.jay.storyguide.plist

Data and logs

The LaunchAgent runtime lives in /Users/jay/.storyguide. Logs are written to storyguide.stdout.log and storyguide.stderr.log in that folder.
